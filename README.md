# 📘 Learning TypeScript

Welcome to my TypeScript learning repository! This project serves as a structured log of my journey to master Type-safe development.

## 🎯 Goal
The goal of this project is to transition from JavaScript to TypeScript and understand how to build more scalable and bug-free applications.

## 🚀 What's Inside?
- **Basics:** Variables, Unions, and Literals.
- **Interfaces & Types:** Defining object shapes and custom types.
- **Functions:** Typed parameters, return types, and arrow functions.
- **OOP:** Classes, Inheritance, and Access Modifiers.

## 🛠️ Tech Stack
- **Language:** TypeScript
- **Environment:** Node.js
- **Compiler:** TSC (TypeScript Compiler)
